In terms of money, there are a few commands that can get you money. Although money is only used for buying colours, it can also be used to convert into things like food, dna and credits by using the !sell command. You buy new dinosaurs with DNA but you also can upgrade them using credits, which can be gained when you level up or purchased for a lot of money. You level up by fighting, and hunting.

If you don’t want to use ! in front of every command, you can use !prefix [new prefix]. Just know that only the owner of a server can use this command.
